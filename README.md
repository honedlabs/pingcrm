# Honed

This repository provides an example project utilising the Honed package ecosystem. The project stack used is:
- Laravel
- Vue 3.5
- Inertia.js 2.0
- TailwindCSS 4.0
- Shadcn/UI for Vue

The Honed packages which are installed are:
- Table
- Refine
- Action
- Lock
- Crumb
- Chart
- Nav
- Modal
